# EventSales AI UI/UX Reference Pack

This pack contains the reference images and markdown guardrails needed to build the EventSales AI MVP UI in Claude Code.

## Folder Structure
- `/reference_images` — approved visual references and per-page mockups
- `/docs` — design system, component rules, page briefs and AI drift guardrails

## Recommended Claude Code Start Prompt
Read all files in `/docs` before implementation. Use `/reference_images/01_dashboard_hero_reference.png` as the primary visual anchor and use the matching page reference image before building each page. Do not drift from the approved shell, colour tokens, page structure or admin/client distinction.

The individual images take priority over the composite image.
