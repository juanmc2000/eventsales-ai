# Reference Images

Use these images as strict visual references when building the EventSales AI UI.

## Primary Visual References

### 00_original_reference.png
Original user-provided inspiration. Use for overall layout structure: dark left navigation, dark top banner, light main page.

### 01_dashboard_hero_reference.png
Primary dashboard reference. Use for the desired hospitality SaaS look: premium, modern, colourful, lively, but still clean.

### 00_composite_all_pages.png
Full page-family concept board. Use to understand how the application pages should feel as a connected product.

## Individual Page References

### 03_Proposals.png

### 04_Pricing_Rules.png

### 07_Insights_Analytics.png
Analytics reference. Use for revenue, conversion, occupancy and lead-time insights.

### 08_Admin_Environments.png
Admin console reference. More utilitarian, technical and infrastructure-oriented.

### 09_Admin_User_Management.png
Internal user management reference. This is not a client-facing page. It is for internal/admin use.

## Guard Rail
Do not introduce unrelated colours, playful illustrations, cartoon mascots, generic CRM blue styling, or heavy grey enterprise UI. The product must remain premium hospitality + commercial intelligence.
