## Icon Style
Use simple outline icons. Recommended libraries:
- Lucide React
- Phosphor Icons
- Heroicons

Do not mix icon families on the same page.

## Hospitality Imagery
Use imagery sparingly:
- Sidebar tenant card
- Restaurant profile pages
- Room/venue asset previews
- Proposal previews

Avoid full-bleed photo backgrounds on operational pages.

## Gradients
Use gradients for:
- active nav item
- KPI icon tiles
- persona cards
- subtle dashboard accents

Do not use gradients for large table backgrounds or form fields.

## Charts
Chart style:
- minimal axes
- soft gridlines
- clear labels
- brand accent colours
- no heavy shadows
- no overly complex chart decorations

Common charts:
- line chart for revenue trend
- funnel for enquiry pipeline
- donut for enquiry source
- bar chart for top venues
- sparkline for KPI cards

## Empty States
Empty states should be concise and useful:
- short title
- one sentence explanation
- primary action
- optional small icon

No cartoon illustrations.

## Loading States
Use skeleton loaders for cards, tables and forms. Avoid large spinners except for full-page initial load.
